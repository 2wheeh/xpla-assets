# Assets

## Layout

```
assets
│
└───cw20
│   │   pairs.dex.js # swap pairs.
│   │   tokens.js # cw20 tokens.
│
└───cw721
│   │   contracts.js # NFT collections.
│   │   marketplace.js # Open NFT marketplaces.
│
└───icon
│   │   extension # for extension.
│   │   game # for game.
│   │   svg # default icons.
│
└───station
│   │    maintenance.js
│
└───chains.js # Xpla chains.
│
└───contracts.js # Xpla contracts.
│
└───extensions.js # Xpla station wallet extension list.
```
